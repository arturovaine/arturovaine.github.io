Conceptual design for Upper limb prosthesis model (TRL-5)

Project Background

This conceptual design represents an upper limb prosthesis developed during a research project focused on creating accessible 3D-printed solutions for people with disabilities.

The project achieved Technology Readiness Level 5 (TRL-5), demonstrating component and/or breadboard validation in a relevant environment.

Read Media Release (Pequenas Empresas & Grandes Negócios)

Key Features

  • 3D-printed components for cost-effective manufacturing
  • Modular design for customization and repairs
  • Designed in SolidWorks with functional mechanics
