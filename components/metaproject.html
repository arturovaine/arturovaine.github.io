A self-aware portrait: seated, looking back with a raised eyebrow, reflected infinitely between mirrors. The recursion mirrors the idea of a metaproject: a portfolio that reflects on itself, so architecture, structure, and identity repeating with intention.

Metaproject

The portfolio itself is a project.

Component Architecture

Modular HTML components loaded dynamically via IntersectionObserver. 17 components organized by priority: header (immediate), critical (immediate), lazy (on scroll).

Data-Driven Rendering

8 JavaScript renderers fetch JSON data and render HTML dynamically. Projects, awards, experience, and more - all managed through structured data files.

Performance First

Lazy loading for images and components. WebP format, skeleton placeholders, optimized aspect ratios for minimal CLS. Videos hosted on Google Cloud Storage.

Dark/Light Theme

Theme switching with localStorage persistence and smooth transitions. Modular CSS architecture split into 5 organized files.
