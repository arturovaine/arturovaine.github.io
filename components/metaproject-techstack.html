Tech Stack

A deliberately minimal stack focused on performance and maintainability.

JS

Vanilla JavaScript

ES6 Modules with native browser APIs. No framework overhead, full control over the DOM.

Why: Maximum performance, zero dependencies, future-proof code.

Tailwind CSS

Utility-first CSS framework loaded via CDN. Rapid prototyping with consistent design tokens.

Why: Fast iteration, no CSS bloat, excellent responsive utilities.

Three.js

3D rendering library for the interactive STL model viewer with OrbitControls.

Why: Industry standard for WebGL, extensive documentation, active community.

Lucide Icons

Beautiful, consistent icon set with 1.5px stroke width. Feather icons fork.

Why: Clean aesthetic, easy to use, actively maintained.

GitHub Pages

Free static hosting with automatic deployment from main branch. Global CDN.

Why: Zero cost, seamless git integration, reliable uptime.

Google Cloud Storage

Video hosting for award highlights. Avoids GitHub's 100MB file limit.

Why: Scalable, cost-effective for media, global edge caching.
